- SQLite를 ARM 리눅스에서 사용할 수 있도록 빌드하기



## 설치 환경
- Ubuntu 18.04
- 타겟: Sabre Lite(i.mx6)
- SQLite-autoconf_3280000

## SQLite 빌드
\# 소스 코드 다운로드<br>
$ wget https://www.sqlite.org/2019/sqlite-autoconf-3280000.tar.gz<br>
$ tar xvzf sqlite-autoconf-3280000.tar.gz<br>
$ cd sqlite-autoconf-3280000<br>
$ mkdir dev

\# --prefix=설치 경로 입력<br>
\# 만약 Windows 환경에서 빌드한다면 Cross Compiler를 지정해야함:   CC=arm-linux-gnueabihf-gcc<br>
$ ./configure --host=arm-linux --prefix=/home/ubuntu/sqlite-autoconf-3280000/dev  

\# 약 10분 정도의 시간이 소요됨<br> 
$ make -j4 

\# 약 5분 정도의 시간이 소요됨<br>
$ sudo make install
